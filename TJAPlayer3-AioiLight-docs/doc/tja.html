.tja フォーマット

更新日: 2019年4月17日

もくじ

  • 仕様
  • 未対応ヘッダ・命令
  • 追加ヘッダ・命令
  • TJAPlayer2 forPCで追加されているもの
  • 例 (サンプル譜面 タイフーンパレード)

TJAPlayer3では、.tja フォーマットのファイルを読み込み譜面を再生することができます。

※各ヘッダ・命令集は現在編集中です。

仕様

  • 推奨される文字コードはShift-JISです。
  • BOM付きUTF-8であればそれも読み込みます。
  • #SCROLL 1.5など、命令と値の間には半角スペースを入れてください。

未対応ヘッダ・命令

以下のヘッダ・命令は未対応です。(多分)使えません。

  • SEVOL:
  • GAME:
  • LIFE:
  • SIDE:
  • TOTAL:

追加ヘッダ・命令

以下のヘッダ・命令は追加されたものです。(他のシミュレータが対応しない限り)他のシミュレータでは無視されるか、正常に再生されなくなります。

ヘッダ・命令名 意味 例
#SENOTECHANGE Ver.1.4.0+ この命令の次の音符のSENotesを変更します。 
#SENOTECHANGE 3
1
#SENOTECHANGE 2
11000000000000,
// ドコドン → コドドン
EXAMn:<条件の種別>,<赤合格値>,<金合格値>,<条件の範囲> Ver.1.5.0+ 段位認定モードにおいて、各条件を設定します。(n=1～3)

条件の種別
g:魂ゲージ
jp:良の数
jg:可の数
jb:不可の数
s:スコア
r:連打数
h:叩けた数
c:最大コンボ数
条件の範囲
m:～以上
l:～未満
 
EXAM1:g,98,100,m
// 条件1:魂ゲージ値 / 98%,100% / 以上
EXAM2:jp,1000,1150,m
// 条件2:良の数 / 1000, 1150 / 以上
EXAM3:jb,10,5,l
// 条件3:不可の数 / 10, 5 / 未満
#NEXTSONG <タイトル>,<サブタイトル>,<ジャンル>,<音源ファイル名>,<初項>,<公差> Ver.1.5.0+ 段位認定モードにおいて、曲を変更しアニメーションを行います。

各パラメータは情報の表示、演奏の進行等に使われます。

タイトル、サブタイトルに半角カンマを使用したい場合、\,と書きます。

 
#START
#NEXTSONG GO!GO!明るい社会,うるまでるび,バラエティ,GO!GO!明るい社会.ogg,560,160
#BPMCHANGE 118.5
#DELAY 2.360
// スタート後、曲名を変更しBPM、オフセットを調節します。
GAUGEINCR:<モード> Ver.1.5.4+ ゲージ増加量の丸め方法を変更します。

モード
NORMAL – 通常の計算方法。
FLOOR – 切り捨てる。
ROUND – 四捨五入する。
CEILING – 切り上げる。
NOTFIX – 丸め処理を行わない。

基本的に、これらのゲージ増加量の大きさは、NORMAL≦FLOOR≦ROUND≦NOTFIX≦CEILINGになります。

 
GAUGEINCR:NORMAL

TJAPlayer2 forPCで追加されているもの

以下のヘッダ・命令はTJAPlayer2 forPCで追加されたもので、TJAPlayer3でも利用可能です。

ヘッダ・命令名 意味 例
GENRE: 譜面のジャンルを指定します。選曲画面での表示、演奏画面などに影響します。

ジャンル一覧
J-POP
アニメ
ボーカロイド
VOCALOID
どうよう
バラエティ
クラシック
ゲームミュージック
ナムコオリジナル

ボーカロイド、VOCALOIDは同義です。

 
GENRE:VOCALOID
// ジャンルをVOCALOIDにする
BGIMAGE: 背景画像を表示します。 
BGIMAGE:hoge.png
// hoge.pngを背景画像として表示
BGMOVIE: 背景動画を表示します。

DirectShow対応のデコーダが必要です。再生できるコーデック、コンテナは各PCによって異なります。

 
BGMOVIE:hoge.wmv
// hoge.wmvを背景動画として表示
MOVIEOFFSET: 背景動画のオフセットを調節できます。

(BGOFFSET:ではありません。)

 
MOVIEOFFSET:1.24
// 背景動画の再生位置を.tjaファイルのOFFSET:から1.24秒ずらす
BALLOONNOR:

BALLOONEXP:

BALLOONMAS:

 譜面分岐時の普通譜面、玄人譜面、達人譜面で使用されるふうせん連打のノルマ数を指定します。 
BALLOONNOR:5,5
BALLOONEXP:8,8
BALLOONMAS:10,10
HIDDENBRANCH: 譜面分岐が行われるまで、譜面分岐が無いように装います。 
HIDDENBRANCH:1
#JPOSSCROLL <時間> <変位> <向き> 判定枠を移動します。

時間
移動に掛ける時間。(秒)
変位
移動の変化量。(px)
向き
0:左 / 1:右
 
#JPOSSCROLL 10 500 1
// 十秒間で右に500px移動させる
